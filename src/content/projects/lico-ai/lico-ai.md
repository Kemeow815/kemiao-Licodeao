这是我的毕业设计 → <strong>基于 AI 的多元定制化视频生成器</strong>，一个全栈项目（目前正在开发中）。

所涉及到的技术栈有：

前端:

- React 全家桶
- TailwindCSS
- Material-UI
- React-Query
- Vite
- TypeScript
- 低代码开发
- FFmpeg 音视频

后端:

- NestJS
- TypeORM
- MySQL
- Redis
- Docker
